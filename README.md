# Gif Searcher

A simple random gif searcher based on search term

![Preview](https://media.giphy.com/media/cXZEkqLN0OoytBQwwi/giphy.gif)

## Instructions
* Ensure you have yarn installed
* `yarn install`
* `yarn start`
* `yarn run test:debug` (to run tests in debug mode)

## Technology Stack

* Vanilla JS
* Parcel Bundler
* Puppeteer
* Stylus
